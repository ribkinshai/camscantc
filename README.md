# 🎥 מערכת מעקב סריקות מצלמות

מערכת לניהול ומעקב אחר סריקות מצלמות במוקד רואה, בנויה ב-Streamlit.

## ✨ פיצ'רים

- **ניהול 200+ מצלמות** - הוספה בודדת או יבוא מרובה
- **מצלמות מרכזיות** - נסרקות בכל שעה עגולה
- **מצלמות בסבב** - 20 מצלמות שונות בכל שעה, בסבב אוטומטי
- **מעקב סריקות** - סימון תקין / תקלה + פירוט אירוע + זיהוי הנציג
- **התראות** - זיהוי אוטומטי של סריקות שהוחמצו
- **מצלמות תקולות** - טבלה נפרדת עם שם, תאריך/שעה, תיאור התקלה
- **משמרות** - בוקר / ערב / לילה (ניתן להתאמה)
- **היסטוריה** - ייצוא נתונים ל-CSV
- **תמיכה מלאה בעברית + RTL**

## 🚀 התקנה והרצה

### דרישות
- Python 3.9+

### התקנה מקומית
```bash
# שכפול הריפו
git clone https://github.com/YOUR_USERNAME/camera-scan-tracker.git
cd camera-scan-tracker

# יצירת סביבה וירטואלית (מומלץ)
python -m venv venv
source venv/bin/activate  # במק/לינוקס
# או ב-Windows:
# venv\Scripts\activate

# התקנת חבילות
pip install -r requirements.txt

# הרצה
streamlit run app.py
```

הדפדפן ייפתח אוטומטית על `http://localhost:8501`

### פריסה ל-Streamlit Community Cloud (חינם)
1. העלה את הפרויקט ל-GitHub
2. היכנס ל-[share.streamlit.io](https://share.streamlit.io)
3. חבר את הריפו וקבע את `app.py` כקובץ הראשי
4. Deploy

> ⚠️ שים לב: ב-Streamlit Cloud, מסד הנתונים לא בהכרח שורד רסטרט של האפליקציה. לפריסת ייצור יציבה מומלץ שרת עם דיסק קבוע.

## 🎯 איך זה עובד

### לוגיקת הסבב
- **מצלמות מרכזיות** - כל 10 המצלמות שסימנת כמרכזיות ייסרקו בכל שעה
- **מצלמות בסבב** - שאר המצלמות מחולקות לקבוצות של 20 (או כפי שתגדיר).  
  בכל שעה נבחרת קבוצה אחרת, כך שהסבב אחיד וניתן לחיזוי.
- **חישוב**: `אינדקס שעה מהעוגן % מס' קבוצות` - מבטיח סבב עקבי.

### התראות סריקות חסרות
המערכת בודקת את 8 השעות האחרונות. אם סריקה מסוימת לא סומנה - היא תופיע ברשימת "סריקות חסרות" בלוח הבקרה. יש זמן חסד (ברירת מחדל: 15 דקות) לפני שהשעה הנוכחית נחשבת לחסרה.

### מצלמות תקולות
מצלמה שדווחה כתקולה **מוחרגת אוטומטית** מלו"ז הסריקות עד שהתקלה תסומן כטופלה.

## 🔧 שימוש ראשוני

1. **הגדרות → הגדרות סריקה** - קבע את מספר המצלמות בסבב (ברירת מחדל: 20) וזמן החסד להתראות
2. **ניהול מצלמות → יבוא מרובה** - הדבק את רשימת שמות המצלמות (שם בכל שורה)
3. **ניהול מצלמות → רשימת מצלמות** - סמן 10 מצלמות כ"מרכזיות"
4. **סריקה שוטפת** - הנציג רואה את המצלמות של השעה הנוכחית וסימן סריקה בוצעה
5. **לוח בקרה** - סקירת מצב + התראות
